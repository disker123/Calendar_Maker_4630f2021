Mobile App Project Update: 12/01/21

Update: Added service worker to allow for install and finished the scripting 
to allow classes to display on the right classes on the right days while also
parsing the academic calendar for monday schedule days and university closed
days.

Sources:
Used: https://youtu.be/WbbAPfDVqfY for setting up the service worker

To run:

To install visit https://john-simonson.github.io/test/ and install the PWA

Once installed:
1.) Add a class via the form
2.) Use the buttons in the header to switch between the form and calendar view
3.) Switch the active day via the navigation buttons on the schedule screen
4.) Can delete entered classes via the delete class button
